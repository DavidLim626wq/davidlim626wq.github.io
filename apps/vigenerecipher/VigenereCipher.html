Vigenere Cipher Demonstration

First described by Giovan Battista Bellaso in 1553, the Vigenere ciphere was used until the time of the American Civil War and once thought unbreakable.

The cipher operates by using a key to encode a plaintext message. A number is assigned to each letter in the key; for example A=0, B=1, and so on. Each letter in plaintext message is shifted by the number of letters indicated by the key, iterating through the key one position at a time.

For example, to encode the message "Hello" using the key "SUN", we would first convert SUN into numerical values 18, 20, and 13. The first letter in the message, H would be shifted by 18 spaces to get Z. Then, since the next letter in the key is U, we would shift 'E" to get Y and so on. When you get to the last letter in the key, you start over. Therefore, the second "L" is encoded using the letter "S" as the key.
